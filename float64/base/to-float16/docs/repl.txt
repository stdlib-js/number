
{{alias}}( x )
    Converts a double-precision floating-point number to the nearest half-
    precision floating-point number.

    Parameters
    ----------
    x: number
        Double-precision floating-point number.

    Returns
    -------
    out: float
        Nearest half-precision floating-point number.

    Examples
    --------
    > var y = {{alias}}( 1.337 )
    1.3369140625
