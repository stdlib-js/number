
{{alias}}( x )
    Converts a half-precision floating-point number to the nearest double-
    precision floating-point number.

    Parameters
    ----------
    x: number
        Half-precision floating-point number.

    Returns
    -------
    out: number
        Nearest double-precision floating-point number.

    Examples
    --------
    > var f16 = {{alias:@stdlib/number/float64/base/to-float16}}( 1.337 )
    1.3369140625
    > var y = {{alias}}( f16 )
    1.3369140625
